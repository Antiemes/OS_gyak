# Próba ZH 2026.03.23. (alacsonyabb szint)

Készítsen shell scriptet a következőkben leírt funkcionalitással pZH1.sh néven. A fájlt lássa el futási joggal.

## Kettes szint

Adott néhány log fájl a `/tmp/logs` könyvtárban. Ezek minden sora egy időpontot, egy MAC címet, egy jelerősséget és egy SSID-t tartalmaz.
A program fogadjon egy paramétert, ami a vizsgálni kívánt fájl neve. (A program dolgozhat csak fájlnévvel, vagy útvonallal együtt
megadott fájlnévvel is. Bármelyik megvalósítás elfogadható.) A program kérjen be egy évet, egy hónapot és egy napot, majd írja ki,
hogy szerepel-e olyan bejegyzés a megadott fájlban, ami erre a napra vonatkozik. Az igen / nem kiírása elég.

A program végezzen hibaellenőrzést is a következőkre:

 * A megadott fájl nem létezik.
 * A bekért dátum egyes mezői túl rövidek, túl hosszúak, vagy nem kizárólag számokból állnak.

Keressünk legalább két olyan teszt esetet, amire igen, illetve legalább egy olyat, amire nem lesz a válasz.

## Plusz egy jegy (A):

A program ne csak egy, hanem tetszőlegesen sok fájlt tudjon megvizsgálni.

## Plusz egy jegy (B)

A program azt is írja ki, hogy az adott napon a vizsgált fájlban mekkora volt a legkisebb és a legnagyobb jelerősség.

## Plusz egy jegy (C)

A program írja ki, hogy milyen SSID-k voltak a vizsgált fájlban a megadott napon.

## Plusz egy jegy (D)

A program az évet, a hónapot és a napot a bekérés helyett az első paraméterből vegye (például 20260302). Ekkor természetesen az első
paramétert nem szabad feldolgozandó fájlnak venni.
